- React

## Como rodar a aplicação :arrow_forward:
No terminal, clone o projeto:
```
git clone https://github.com/Guihnovo/Portfolio.git
```
Entre na pasta do arquivo
```
cd Portfolio/src
```
Instale as dependências
```
npm install
```
Execute a aplicação
```
npm run start
```
Abrir o navegador no localhost:3000
